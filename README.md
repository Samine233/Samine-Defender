# 关于Samine Defender
## 开源协议
本软件开源遵循**GPL v2**协议，一切未经允许的再加工和仿制均视为违反协议。下载并编译源代码默认视做您已经同意本协议。
## 防御原理
通过windows提供的函数和API实现对部分程序的窗口、进程名进行持续监控，如精确查找发现类似程序，则会调用API清理线程。同时程序通过索取管理员权限，来进行高自由度的system操作。可以方便地结束联想云教室的进程树。

本软件索取管理员权限仅用于删除进程，不会对系统和您的个人数据进行任何读写，也不会将您的个人信息上传至云端。

# Q&A
## Q：我的Samine Defender在任务栏找不到了怎么办？我该如何删除Samine Defender？
## A: Ctrl + Shift + Esc打开任务管理器，找到Defender.exe，将其删除。
## Q:我的Samine Defender的重复打开功能失效了/提示未找到Defender.exe怎么办？
